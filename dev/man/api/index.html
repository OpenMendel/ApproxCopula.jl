fit!(gcm::NBCopulaVCModel, solver=Ipopt.IpoptSolver(print_level=5))

Fit an NBCopulaVCModel object by block MLE using a nonlinear programming solver. Start point should be provided in gcm.β, gcm.Σ, gcm.r. In our block updates, we fit 10 iterations of gcm.β, gcm.Σ using IPOPT, followed by 10 iterations of Newton on nuisance parameter gcm.r. Convergence is declared when difference of successive loglikelihood is less than tol.
