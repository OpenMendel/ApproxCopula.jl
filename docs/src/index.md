## Installation

Download and install [Julia](https://julialang.org/downloads/). Within Julia, copy and paste the following: 
```julia
using Pkg
pkg"add https://github.com/sarah-ji/GLMCopula.jl"
```
This package supports Julia `v1.6`+.
